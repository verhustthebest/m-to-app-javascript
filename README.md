# Weather App

A simple weather application built with JavaScript using the OpenWeather API.

## Features

- Search weather by city
- Display temperature
- Display humidity
- Display wind speed
- Display weather icon

## Technologies

- HTML
- CSS
- JavaScript
- OpenWeather API

## How to use

1. Enter a city name
2. Click "Recherchez" or press Enter
3. Weather information will appear
